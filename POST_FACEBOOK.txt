Grille Lanceur v2 — votre lanceur sur le bureau (Linux Mint / Cinnamon)

Vous en avez assez du menu pour tout retrouver ? Ce desklet affiche vos applis et raccourcis directement sur le bureau : en grille d’icônes ou en liste avec les noms.

Ce que vous pouvez faire :
• Glisser une app depuis le menu Cinnamon → elle s’ajoute tout de suite
• Glisser une icône du bureau (Nemo) → copie dans le lanceur (l’icône reste sur le bureau)
• Clic gauche → lancer
• Clic droit sur une icône → « Retirer l’élément »
• Clic droit sur le desklet → renommer, trier par nom, passer en mode liste ou grille, ajouter/retirer colonnes et lignes
• Mode liste : icône + nom sur chaque ligne (idéal si vous préférez lire les noms)
• Plusieurs desklets possibles : une grille « Travail », une « Jeux », etc.

Installation en 3 étapes :
1) Copier le dossier grid-launcher-v2@local dans ~/.local/share/cinnamon/desklets/
2) Alt+F2 → r (recharger Cinnamon)
3) Gestionnaire de desklets → Grille Lanceur v2 → Ajouter au bureau

Mode d’emploi complet (installation, menus, dépannage) : fichier MODE_EMPLOI_DETAILLE.txt dans le dossier du desklet, ou sur GitHub si vous publiez le projet.
