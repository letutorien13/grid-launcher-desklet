================================================================================
  GRILLE LANCEUR v2 — Desklet Cinnamon
  Résumé pour GitHub (mode d'emploi complet : MODE_EMPLOI_DETAILLE.txt)
================================================================================

  Desklet de raccourcis sur le bureau : mode grille (icônes) ou mode liste
  (icône + nom). Remplissage par glisser-déposer, redimensionnement par menu.

  Documentation complète : voir MODE_EMPLOI_DETAILLE.txt (20 sections).


--------------------------------------------------------------------------------
  INSTALLATION
--------------------------------------------------------------------------------

  1. Copier grid-launcher-v2@local dans ~/.local/share/cinnamon/desklets/
  2. Alt+F2 → r
  3. Gestionnaire de desklets → Grille Lanceur v2 → Ajouter au bureau


--------------------------------------------------------------------------------
  UTILISATION RAPIDE
--------------------------------------------------------------------------------

  Ajouter   : menu Cinnamon ou icône bureau → glisser sur le desklet
  Lancer    : clic gauche
  Retirer   : clic droit sur l'icône → Retirer l'élément
  Desklet   : clic droit hors icône → Renommer, Trier, Liste/Grille, Colonnes/Lignes
  Données   : ~/.config/cinnamon_grid_launcher_data.json
